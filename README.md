# 🚗 Thomas Travels - Driver Management System

## 📌 Project Overview

This is a simple Java-based application developed to manage driver details for a travel service company.

The system allows:

* Storing driver information
* Searching drivers by ID
* Filtering drivers by category
* Finding drivers with maximum travel distance

---

## 🧱 Tech Stack

* Java
* OOP Concepts
* ArrayList (Collections Framework)

## 🚀 Features

### 1. Check Car Driver

Checks whether a given driver belongs to the "Car" category.

### 2. Retrieve Driver by ID

Returns driver details using driver ID.

### 3. Count Drivers by Category

Counts number of drivers for a given category (Car/Auto/Lorry).

### 4. Retrieve Drivers by Category

Returns list of drivers belonging to a specific category.

### 5. Maximum Distance Travelled

Finds the driver who has travelled the maximum distance.
